# Cricket Heroes Frontend

A React application built with Vite and Tailwind CSS for managing cricket match inputs.

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Features

- React 18 with Vite
- Tailwind CSS for styling
- IPL 2022 Points Table stored in localStorage
- Responsive design
- Form validation

## Technology Stack

- React 18.2.0
- Vite 5.0.8
- Tailwind CSS 3.4.0
